# Rspec

# Installation

```bash
gem install 'rspec-rails'
rails generate rspec:install
```

This will generate the `spec` folder, `rails_helper.rb`, `spec_helper.rb` files.

## Run tests

```bash
cd myproject
bundle exec rspec
```
